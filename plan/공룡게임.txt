이름: 크롬 공룡게임 만들기
참가자:문종훈, 고경민

개발 방식: pygame 사용하여 공룡의 조작과 장애물과의 충돌,점수를 구현함


= 기록 =
1주차
- 기획
	화면 띄우고 공룡 이미지 구현

- 실제로 한 것
		pygame을 이용해 게임 화면을 띄우는 것을 해보았고, 		공룡 이미지를 화면 내에 나타나게 하는 작업을 진행했		는데, 이미지 파일의 경로를 설정하는것이 조금 어려웠		으나 인터넷에서 파일 경로를 설정하는 방법에 대해 검		색해 절대 경로가 아닌 상대 경로로 파일을 찾아내는 방		법으로 해결했다.

2주차
- 기획
	공룡 이미지의 동작 구현
- 실제로 한 것
		와이파이 이슈로 작업을 못했음.


3주차
- 기획
	점수와 장애물 충돌 구현 + 2주차 작업까지 완료
- 실제로 한 것
	class를 이용해 공룡과 장애물의 객체를 정의했고, 장애물이 나오는 속도와 간격 조절을 위해 random을 통해 타이머를 만들어 조절함. 그리고 점수가 100점이 오를때마다 속도가 올라가게 했음.
공룡을 이미지를 불러와 사용하는건 어려워서 파이썬에서 직접 만들어 사용하는 방향으로 바꿈
